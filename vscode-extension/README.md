# OpenAs3D - 3D Codebase Explorer

![GitHub Release](https://img.shields.io/github/v/release/bitkojine/openas3d?label=latest%20release&color=blue)

Visualize your codebase dependencies and architecture in navigable 3D worlds directly within Visual Studio Code.

## Features

- **3D Visualization**: Explore your codebase as an interactive 3D world
- **Dependency Mapping**: See file relationships and dependencies at a glance
- **Architecture Insights**: Understand your project structure through spatial visualization
- **Interactive Navigation**: Move through your code in 3D space
- **Real-time Updates**: Changes to your codebase are reflected immediately

## Usage

### Commands

- `OpenAs3D: Explore Dependencies in 3D` - Analyze and visualize dependencies
- `OpenAs3D: Open as 3D World` - Open the current file in the 3D world

### Getting Started

1. Install the extension from the VSCode Marketplace
2. Open a project folder
3. Use one of the commands above to launch the 3D visualization
4. Navigate through your codebase using mouse/touch controls

## Controls

- **Mouse**: Click and drag to rotate view
- **Scroll**: Zoom in/out
- **Click files**: Select and inspect
- **Right-click**: Context menu with additional options
